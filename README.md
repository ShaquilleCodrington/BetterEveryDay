# Better Every Day

Better Every Day is a React + TypeScript productivity tracker focused on helping users choose a mental mode, select an activity, and log focused sessions.

## Current Features

- Focus mode selection
- Activity buttons based on selected focus mode
- Activity page view
- Persistent notes using localStorage
- Back navigation to focus screen

## Tech Stack

- React
- TypeScript
- Vite
- localStorage

## Next Features

- Adjustable countdown timer
- Start/save session history
- View old sessions
- Navigation layout
